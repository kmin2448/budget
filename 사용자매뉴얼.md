## 1. 시스템 접속 방법

1. 웹 브라우저(Chrome 권장)를 열고 시스템 주소로 접속합니다.
2. **"Google로 로그인"** 버튼을 클릭합니다.
3. 조직 Google 계정(예: `xxxx@knu.ac.kr`)으로 로그인합니다.
4. 처음 로그인 시 Google이 권한 허용 팝업을 표시합니다 → **"허용"** 클릭.

> **중요:** 로그인 후 아무 메뉴도 보이지 않는다면, 관리자에게 계정 등록을 요청하세요.  
> 시스템에 미리 등록된 계정만 접속이 가능합니다.

## 3. 주요 기능 사용법

### 3-1. 대시보드

**볼 수 있는 것**
- 전체 예산 대비 집행률 요약 카드
- 프로그램별 예산계획 / 집행완료 / 집행예정 금액 테이블
- 비목별 집행 현황

**프로그램 순서 변경**
1. 프로그램명 왼쪽의 ⠿ (점 6개) 아이콘을 마우스로 클릭한 채 드래그합니다.
2. 원하는 위치에 놓으면 순서가 바뀝니다.

---

### 3-2. 비목별 집행내역

**탭 전환:** 화면 상단 탭(인건비 / 장학금 / 교육연구프로그램개발운영비 …)을 클릭해 비목을 선택합니다.

**집행 건명 입력**
1. **"+ 항목 추가"** 버튼 클릭
2. 건명, 금액, 집행 월, 담당자 등 입력
3. **"저장"** 클릭

**PDF 증빙 업로드**
1. 집행 항목 오른쪽의 📎 아이콘 클릭
2. PDF 파일 선택 (PDF 형식만 가능)
3. 업로드 완료 후 파일 아이콘이 초록색으로 변하면 정상입니다.
4. 아이콘 클릭 시 Google Drive에 저장된 파일이 새 탭으로 열립니다.

**집행 월 이동 (드래그)**
- 항목 왼쪽의 ⠿ 아이콘을 드래그해 다른 월 그룹으로 이동할 수 있습니다.

> ⚠️ **편집 권한이 없으면** 입력 버튼이 보이지 않습니다. 관리자에게 권한 요청을 하세요.

---

### 3-3. 예산관리

**예산표 조회**
- 비목별 편성액 / 증감액 / 조정 후 금액 / 집행완료 / 집행예정 / 잔액 / 집행률을 확인합니다.

**증감액 수정**
1. 수정하고 싶은 세목의 **증감액** 칸을 클릭해 금액 입력
2. **"저장"** 버튼 클릭 → Google Sheets에 즉시 반영

**예산표 PDF 출력**
1. 오른쪽 상단 **"PDF 출력"** 버튼 클릭
2. 브라우저 인쇄 창이 열리면 **"PDF로 저장"** 선택

**변경 이력 조회**
- 하단의 **"변경 이력"** 탭에서 누가 언제 증감액을 수정했는지 확인할 수 있습니다.

---

### 3-4. 선지원금

- 선지원금 지급 내역 입력 / 정산 처리
- 상단 **"+ 추가"** 버튼으로 새 항목 입력
- 정산 완료된 항목은 **"정산"** 버튼 클릭

---

### 3-5. 산단카드

- 산단 법인카드 사용 내역 관리
- **"+ 내역 추가"** 버튼으로 사용 일자 / 금액 / 내용 입력
- 목록에서 항목 클릭 시 수정 가능

---

### 3-6. 권한관리 (슈퍼어드민 전용)

> 이 메뉴는 **슈퍼어드민** 계정으로만 접근할 수 있습니다.

**사용자 추가 (개별)**
1. **"사용자 추가"** 버튼 클릭
2. 이메일 주소 입력, 이름·역할 선택
3. **"추가"** 클릭 → 다음 번 해당 이메일로 로그인 시 접속 가능

**사용자 일괄 추가 (엑셀)**
1. **"양식 다운로드"** 버튼으로 Excel 양식 받기
2. 양식의 첫 번째 행(헤더)은 수정하지 말고, 2행부터 정보 입력
3. 저장 후 **"엑셀로 일괄 추가"** 버튼으로 파일 업로드

**역할 변경**
- 사용자 행의 역할 드롭다운을 클릭해 `뷰어 / 어드민 / 슈퍼어드민` 중 선택

**세부 편집 권한 부여 (어드민 전용)**
- 역할이 **어드민**인 사용자 행에서 허용할 기능 체크박스에 체크
- 뷰어/슈퍼어드민은 세부 권한 체크박스가 비활성화 상태로 표시됨

**사용자 삭제**
- 해당 행 오른쪽 🗑️ 아이콘 클릭 → 확인 팝업에서 **"삭제"** 클릭

**순서 변경**
- 각 행 왼쪽의 ⠿ 아이콘을 드래그해 표시 순서 변경 (저장은 되지 않으며 화면 내 정렬 목적)

## 4. 자주 묻는 질문 (FAQ)

**Q. 로그인했는데 아무 메뉴도 보이지 않아요.**  
A. 슈퍼어드민에게 연락해 해당 구글 계정이 시스템에 등록되어 있는지 확인을 요청하세요.

**Q. 집행내역을 입력하려는데 버튼이 없어요.**  
A. 현재 계정에 해당 비목의 **집행내역 편집 권한**이 없습니다. 슈퍼어드민에게 권한 부여를 요청하세요.

**Q. PDF를 업로드했는데 파일이 어디에 저장되나요?**  
A. 업로드한 사람의 **개인 Google Drive > COSS_지출부 > {비목명}** 폴더에 저장됩니다.

**Q. 예산표 금액과 집행내역 금액이 다르게 보여요.**  
A. Google Sheets가 직접 수정된 경우 발생할 수 있습니다. 개발자에게 Sheets 데이터 확인을 요청하세요.

**Q. 로그인은 됐는데 파일 업로드가 안 돼요.**  
A. 로그아웃 후 재로그인하면 해결되는 경우가 많습니다. → [업로드 오류 대처법](#drive-업로드-오류) 참고

## 5. 트러블슈팅 — 오류 상황별 대처법

### 로그인 오류

| 증상 | 원인 | 해결 방법 |
|------|------|-----------|
| 로그인 버튼 클릭 후 흰 화면 | 브라우저 캐시 문제 | `Ctrl+Shift+Delete` → 캐시 삭제 후 재시도 |
| "액세스 차단됨" 메시지 | 미등록 계정 | 슈퍼어드민에게 계정 등록 요청 |
| 로그인 루프 (계속 로그인 화면으로 돌아옴) | 쿠키 차단 설정 | 브라우저 설정에서 서드파티 쿠키 허용 후 재시도 |

---

### Drive 업로드 오류

**증상:** "Google Drive 접근 권한이 없습니다. 로그아웃 후 다시 로그인해 주세요."

**원인:** 로그인 세션의 Drive 접근 토큰이 만료되었습니다.

**해결 방법:**
1. 오른쪽 상단 계정 아이콘 → **"로그아웃"**
2. 다시 로그인 → 구글 권한 허용 팝업에서 **"허용"** 클릭
3. 업로드 재시도

---

**증상:** "PDF 파일만 업로드 가능합니다."

**해결 방법:** 파일 확장자를 `.pdf`로 변환 후 업로드합니다.
- Word/Excel → `파일 > 다른 이름으로 저장 > PDF` 선택

---

**증상:** 업로드 시 "파일 크기 초과" 메시지

**해결 방법:** 파일 크기가 50MB를 넘지 않는지 확인합니다.  
PDF 용량 줄이기: [Smallpdf](https://smallpdf.com/compress-pdf) 등 온라인 압축 도구 사용

---

### 예산 데이터 조회 오류

**증상:** 예산관리 / 대시보드 화면에서 "데이터를 불러오지 못했습니다" 또는 빈 화면

**체크 순서:**
1. 인터넷 연결 확인 (다른 사이트 접속 테스트)
2. 페이지 새로고침 (`F5`)
3. 그래도 안 되면 → **Google Sheets가 열려 있는지 확인** (다른 사람이 편집 중이면 잠시 기다릴 것)
4. 계속 안 되면 개발자에게 아래 정보와 함께 연락
   - 오류가 발생한 메뉴명
   - 오류 발생 시각
   - 화면 캡처

---

### 저장이 안 되는 경우

**증상:** 예산 증감액 / 집행내역 입력 후 저장 클릭해도 변경이 반영되지 않음

**체크 순서:**
1. **편집 권한** 확인 — 현재 계정에 해당 메뉴의 편집 권한이 있는지 슈퍼어드민에게 확인
2. **Google Sheets 연동 상태** 확인 — Sheets가 접근 가능한지 확인 (구글 서버 장애 시 [Google 상태 페이지](https://workspace.google.com/status) 확인)
3. 로그아웃 후 재로그인 후 재시도

---

### 화면이 깨지거나 레이아웃이 이상한 경우

**해결 방법:**
1. 브라우저 확대/축소 배율 확인 → `Ctrl+0` (기본값 100%)으로 초기화
2. Chrome 사용 권장 (IE, 구버전 Edge는 지원하지 않음)
3. 브라우저 캐시 삭제: `Ctrl+Shift+Delete` → "캐시된 이미지 및 파일" 체크 → 삭제

---

### 권한관리 페이지가 보이지 않는 경우

**원인:** 현재 계정의 역할이 `슈퍼어드민`이 아닙니다.  
**해결 방법:** 슈퍼어드민 계정으로 로그인하거나, 슈퍼어드민에게 역할 변경을 요청합니다.

## 6. 긴급 연락 / 개발자 전달 사항

오류가 해결되지 않으면 개발자에게 아래 내용을 함께 전달해 주세요.

```
1. 오류 발생 메뉴: (예: 예산관리 > 증감액 저장)
2. 오류 발생 시각: (예: 2026-04-19 14:32)
3. 화면에 표시된 오류 메시지 전체: (캡처 첨부)
4. 시도한 해결 방법:
5. 브라우저 종류 및 버전: (Chrome 주소창에 chrome://version 입력)
```

**브라우저 개발자 도구 오류 로그 캡처 방법 (개발자 요청 시)**
1. `F12` 키 누름
2. 상단 탭에서 **Console** 클릭
3. 빨간 글씨로 표시된 오류 메시지 캡처해서 전달
